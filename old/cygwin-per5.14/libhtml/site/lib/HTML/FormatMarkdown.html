NAME

HTML::FormatMarkdown - Format HTML as Markdown

Back to Top

VERSION

version 2.11

Back to Top

SYNOPSIS

    use HTML::FormatMarkdown;
    my $string = HTML::FormatMarkdown->format_file(
        'test.html'
    );
    open my $fh, ">", "test.md" or die "$!\n";
    print $fh $string;
    close $fh;

Back to Top

DESCRIPTION

HTML::FormatMarkdown is a formatter that outputs Markdown.

HTML::FormatMarkdown is built on the HTML::Formatter manpage and documentation for that module applies to this - especially new in the HTML::Formatter manpage, format_file in the HTML::Formatter manpage and format_string in the HTML::Formatter manpage.
