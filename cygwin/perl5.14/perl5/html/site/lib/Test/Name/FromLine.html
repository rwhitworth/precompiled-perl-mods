NAME

Test::Name::FromLine - Auto fill test names from caller line

Back to Top

SYNOPSIS

  use Test::Name::FromLine; # just use this
  use Test::More;
  is 1, 1; #=> ok 1 - L3: is 1, 1;
  done_testing;

Back to Top

DESCRIPTION

Test::Name::FromLine is test utility that fills test names from its file. Just use this module in test and this module fill test names to all test except named one.
